Component
App

It uses Dart as the programming language, and the developer only needs to create one codebase to build apps for Android and iOS. Our app needs to be able to run on both Android and iOS platform, and using Flutter to develop can reduce the development effort significantly. Flutter has many plugins developed by third party, which implements many necessary functions for our project, for example, WiFi scanning, Speech to Text, Text to Speech, Push Notification and HTTP Connection.

Component
Orgnisation Server

Organisation server is a web app for IT administrator to view all the profiles of local servers. It lists all devices’ status and location of each local Sight++ setting for monitoring. It provides API for local servers for local devices update. Also, after administrator set up the knowledge base of location in Azure QnA Maker, administrator can save the link of it in organisation server so that once they want to make modification to knowledge base, it can easily be accessed through organisation server page.
